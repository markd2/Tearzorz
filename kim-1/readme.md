# KIM-1 stuff

Building a PAL-1, which runs KIM-1.

Found some ROM images at http://retro.hansotten.nl/6502-sbc/kim-1-manuals-and-software/kim_1-roms/

* 6530-002 implements a TTY interface, a keyboard interface (hence the name Keyboard Interface Monitor) and 6 7 segment LED displays.
* 6530-003 is an audio cassette recorder extension of the KIM monitor
